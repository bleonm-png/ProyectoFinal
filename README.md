# Sistema de Gestión de Ventas e Inventario

## Descripción

Proyecto desarrollado en C++ para la administración de productos, ventas e inventario de una tienda mediante consola.

El sistema utiliza programación estructurada, modularización y archivos binarios para almacenar información.

---

#  Funcionalidades

##  Gestión de Productos

* Registrar productos
* Listar productos
* Buscar por código
* Buscar por nombre
* Actualizar stock
* Modificar precio
* Desactivar productos

---

##  Proceso de Ventas

* Crear ventas
* Agregar múltiples productos
* Buscar productos por código o nombre
* Validar stock
* Aplicar IVA y descuentos
* Actualizar inventario automáticamente
* Guardar ventas en archivos binarios

---

##  Reportes

* Productos con menor stock
* Productos más vendidos
* Ventas totales del día
* Ventas por mes
* Ordenamientos por:

  * precio
  * stock
  * ventas acumuladas

---

##  Utilidades

* Exportar reportes a .txt
* Reiniciar inventario
* Estadísticas generales

---

#  Tecnologías Utilizadas

* C++
* Archivos binarios (.dat)
* Programación estructurada
* Modularización mediante archivos `.cpp` y `.h`

---


#  Características Implementadas

* Uso de `struct`
* Archivos binarios
* Matriz bidimensional
* Bubble Sort
* Selection Sort
* Validaciones y manejo de errores
* Uso de `try/catch`
